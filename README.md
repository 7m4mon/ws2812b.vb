# ws2812b.vb
Controll WS2812B by VisualBasic.NET (SerialPort 3Mbps 7-n-1)

This module controls WS2812B with 3M Baud UART. (ex. FT234X)

3M baud対応のUARTモジュール（秋月のAE-FT234X等）を使ってWS2812BをVB.netから制御するモジュール。
